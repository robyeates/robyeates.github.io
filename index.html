Cloud computing with containers
has changed the way we create software

I'm enormously excited to work in an industry that is undergoing huge changes to the way we think about, design and build server side software.

I'd like to highlight some of the tools I've used and projects I've delivered that showcase a number of the principals of cloud computing.

How has Cloud computing improved Software Development

Horizontal Scalability allows for a distributed application to add more nodes to meet demand. I've used this at Goldman Sachs with a simple DNS implementation and at Gresham utilising ECS and auto-scaling groups to provision more nodes on demand. With Docker images containing the artifacts this has been a reliable mechanism to allow us to meet demand.

Compute on demand means that we don't need any more expensive bare metal boxes sitting in a rack with 0.1 load. Using a AWS Node Swarm I delivered a build chain that requested build nodes when required and released them once complete.

Infrastructure as code allows Developers and Operations to work more closely together. Using CloudFormation I delivered a Stack that could be recreated anywhere easily. Ensuring repeatability and reliability of testing and production deployment.
